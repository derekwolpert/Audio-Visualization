Welcome to vi·si·cal·i·ty (visual + musicality)!


vi·si·cal·i·ty is a web-based music visualizer utilizing Web Audio API and D3.js to deliver an interactive, customizable user experience.


Note: For best compatibility please use vi·si·cal·i·ty with a desktop version of Google Chrome. With other web browsers some features may not be available due to varying support of Web Audio API methods. While this project's UI / UX is optimized for desktop browsers, it is also compatible with mobile devices that support Web Audio API.


Features:


  • Select an audio or video file stored locally on your device to use with the visualizer.
  • Seamlessly switch between nine visualizer options (visualizer 8 and 9 utilize a Web Audio API method, getFloatTimeDomainData, that is not compatible with Safari and Internet Explorer).
  • Select from nine diverse color schemes for visualizer.
  • Endless background color gradient customization! Select any three colors for the background gradient, set the gradient's direction, and watch the website's favicon change accordingly.
  • Control the playback of your audio - view file's title, total length and current position.
  • Change the gain of the audio, and watch as the intensity of the visualizer correspondingly adjusts.
  • Quick access to transition in and out of full-screen mode.
  • Intuitive UI / UX that automatically hides when visualizer audio is playing and there is no detected mouse movement.
  • Flexible combination of onscreen and keyboard controls.
